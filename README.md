# Teacher/Admin Portal

A comprehensive teacher portal for managing classes, grades, attendance, and student performance.

## 🚀 Features

### 👨‍🏫 Teacher Dashboard (HIGH IMPACT)
- **Overview Cards**: Total Classes (5), Students (127), Pending Grades (12), Today's Classes (3)
- **Today's Schedule**: Visual class cards with times, rooms, and student counts
- **Quick Actions**: Enter Grades, Take Attendance, Post Announcements, Generate Reports
- **Notifications Panel**: Grade deadlines, admin announcements, meetings

### 📚 Class Management
- **My Classes**: List of assigned classes with student counts and attendance stats
- **Class Overview**: Student enrollment, attendance rates, pending grades per class
- **Quick Actions**: View students, take attendance, enter grades

### 📝 Grade Management (CORE FEATURE)
- **Inline Editing**: Click any grade cell to edit instantly
- **Auto Calculation**: Final grades calculated automatically
- **Grade Status**: Color-coded performance indicators (Excellent/Very Good/Good/Needs Attention)
- **Bulk Operations**: Save all changes, export grades
- **Real-time Updates**: Changes saved instantly

### 📅 Attendance System
- **One-Click Attendance**: Mark Present/Absent/Late/Excused
- **Time Tracking**: Automatic timestamps
- **Notes System**: Add notes for special cases
- **Export Reports**: Generate attendance reports

### 👨‍🎓 Student Performance
- **Individual Tracking**: GPA, attendance, progress visualization
- **Performance Indicators**: Color-coded status cards
- **At-Risk Alerts**: Students needing attention
- **Progress Monitoring**: Visual progress bars

### 📢 Announcements
- **Class-targeted**: Post to specific classes or all classes
- **Rich Content**: Title, message, target audience
- **Timeline**: Recent announcements with timestamps

### 📊 Reports & Analytics
- **Grade Distribution**: Charts and statistics
- **Attendance Summary**: Class and individual attendance rates
- **Performance Trends**: Top performers and at-risk students
- **Export Options**: PDF and Excel reports

## 🚀 Deployment

This portal can be deployed as a separate Heroku app alongside your Student Portal:

1. Create new Heroku app: `your-teacher-portal`
2. Add JawsDB MySQL add-on
3. Deploy the `admin/` directory
4. Run database setup script
